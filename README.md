# RO Unlimited — Contractor & Developer

Professional website for RO Unlimited Contractor & Developer.
Built with Next.js 14, Tailwind CSS, and GSAP (Phase 2).

## Brand
- **Colors:** Black (#1A1A1A), Gold (#C9A84C), White
- **Fonts:** Oswald (headings), Barlow (body), JetBrains Mono (accents)
- **Theme:** Construction site that builds itself as you scroll

## Division Structure
- **Home** — Company overview, division selector, trust stats
- **Residential** — Custom homes, luxury builds, renovations
- **Commercial** — Steel builds, storefronts, development
- **Land Grading** — Excavation, site prep, foundation
- **The Build Process** — Phase-by-phase timeline
- **Contact** — Lead capture form

## Setup
```bash
npm install
npm run dev
```

## Build Phases
- **Phase 1** (current): Full structure, all pages, brand system, GSAP hooks
- **Phase 2**: GSAP ScrollTrigger animations, construction build effects
- **Phase 3**: Polish, particles, loading sequences, micro-interactions
- **Phase 4**: CMS integration, photo galleries, form backend
